Browsers

A web browser is an application that lets you navigate the internet. This is how you will access websites and online services. By using a browser, you can search for information, shop online, connect with others, and countless other things.

Navigation Menus

Navigation menus are an important part of using a website. They provide a way to quickly move between different sections and pages. There are a few different ways that navigation menus can look and be used.

Example

Example

Example

Top Nav

This menu is located at the top of the webpage and typically has the most important sections of the website. This menu can include other drop-down options within it for more specific subcategories.

Example

Example

Example

Side Nav

This type of navigation is very similar to a top navigation bar but is located on the side of the webpage and can be used to go through different parts or sections of a website. This type of menu is usually used to show a lot of similar content or on websites with multiple levels of menus.

Collapsed Nav

This type of menu is often hidden away until you try to use it. This menu tends to look like three horizontal lines stacked onto each other that can be clicked to show navigation options. Collapsed navigation icons tend to be in the upper corners of a webpage and are very common on mobile devices.

Common Browser Elements

When browsing the internet there are some things that can help you navigate easier. Knowing what they look like and do will help you become more familiar and comfortable with any website.

Search Bar

The search bar is a text box that lets you search a website or the internet for information. Typically the textbox will have a magnifying glass as an icon.

This is an example of a scrollbar! Usually they're vertically aligned and on the side of the webpage but in this case it's a horizontal scrollbar for text!

Scroll Bar

A scroll bar is a visual representation of how far down a webpage you are. They are typically shown as a thin dark colored bar.

Tabs

Many browsers use tabs to help moving back and forth between multiple webpages quick and easy.

Arrows

These are the navigation arrows. The back arrow will take you back to the previous webpage you were on if possible. The forward arrow will return you to the page you were on before.

Bookmarks

Bookmarks are a very useful feature of browsers that allow you to remember web pages you like to go to often.

Refresh

This is a refresh button, it can be found on most browsers, and it is usually shown by an arrow circling back on itself.
